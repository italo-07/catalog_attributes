Selection Attributes Catalogue

About

This catalog presents the selection attributes used to support approaches of testing techniques selection proposed in the literature. In a previous work [Santos et al.], a mapping study was conducted to identify the selection attributes used in the process of testing techniques selection. The aim of this catalog is to present to testers practioners and the software testing scientific community what these attributes are and how they can be used according to the application domain that a testing technique is intended to select.
