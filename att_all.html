Selection Attributes Catalogue

All Categories

Categories Definition:

→ General Testing Characteristics: this category includes attributes concerned about testing technique characteristics, such as application cost or how easy the testing technique is to understand;

→ Tools: this category contains information regarding the tools available for a testing technique;

→ Test Team: this category includes information related to test team characteristics;

→ Software Project: this category is composed of attributes related to information regarding earlier projects in which the testing technique is applied;

→ ODC Faults: this category presents attributes that identify orthogonal defect classification faults;

→ Web Applications: this category is composed of attributes used in the web application domain;

→ Concurrent Software: this category is composed of attributes used in the concurrent software domain;

→ Model-Based Testing: this category is composed of attributes used in the context of model-based testing.
